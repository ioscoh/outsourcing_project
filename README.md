아웃소싱 프로젝트

# 📌프로젝트 소개

***
기업용 태스크 관리 시스템 (TaskFlow)의 백엔드 개발

## 💡핵심 기능

***
1. 회원가입 / 로그인 : 회원 CRUD, 로그인, 회원탈퇴
2. 태스크 관리 : 태스크 CRUD(Soft Delete 처리), 상태 변경
3. 댓글 관리 : 댓글 CRUD(검색 기능)
4. 대시보드 : 통계 정보 제공, 태스크 요약
5. 활동로그 : 활동 기록, 저장, 조회

## 🔧사용하는 기술
`Java` `Spring Boot` `Github` `MySQL` `JPA` `JWT`
